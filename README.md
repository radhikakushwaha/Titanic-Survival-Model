# Titanic Survival Prediction App

A machine learning–based web application that predicts passenger survival during the Titanic disaster using historical data. 
The app is built with **Streamlit** and uses a **Logistic Regression** model.

## Features

* Real-time survival prediction
* Interactive Streamlit interface
* Data preprocessing and feature encoding
* Logistic Regression–based classification

## Tech Stack

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn

## How to Run


git clone https://github.com/your-username/titanic-survival-prediction.git
cd titanic-survival-prediction
pip install -r requirements.txt
streamlit run app.py
